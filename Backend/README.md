## Installation GUIDE
### clone the project
```
  git clone git@github.com:Ankur-Ydv/GETITDONE-Backend.git
```
#### go to the project directory
```
  cd Getitdone-backend
```
#### Install dependencies
```
  npm install
```
#### Connect Local Server
```
  using Mongo Atlas/Compass
```
#### start the app
```
  npm start
```
